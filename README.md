# Even-Odd Number Analyzer
This project separates even and odd numbers entered by the user.

Built using:
- HTML
- CSS
- JavaScript (arrays, loops, DOM manipulation)

This project allows the user to input multiple numbers separated by commas.
When the user clicks the "Analyze Numbers" button, the program separates even and odd numbers.
The results are displayed dynamically in the center of the page.
Even numbers appear in green and odd numbers appear in red, using JavaScript styling.
Features
Input field for numbers
Button to trigger analysis
Dynamic display of results
Highlighting of even and odd numbers (JS controlled)
Clean and simple layout (input/button centered, background neutral)
How to Use
1. Type numbers separated by commas in the input field. Example: 1, 2, 3, 4, 5, 6
2. Click the "Analyze Numbers" button.
3. See the results in the center:
   . Even numbers in green
   . Odd numbers in red
Techniques Used
. JavaScript: for loop, arrays, push, event listener, DOM manipulation, .trim()
. HTML: input, button, <p> for results
. CSS: centering the input/button and basic layout
. JS dynamic styling for coloring the resuls
